## 🎮 미니게임천국

<div align="left">

**📅 개발 기간:** 2023.10.18 ~ 2023.12.13  
**👥 인원 구성:** 4명  
**🎯 담당 역할:** 사용자 인터페이스 및 디자인 작업, 발표 준비, 게임 기능 테스트 및 오류 수정, 팀 내 기능 연동 및 전체 시스템 검토 (25%)  
**🏆 수상 내역:** SW 경진대회 대상 (2023, 남서울대학교 컴퓨터소프트웨어학과)

### 📝 프로젝트 개요

C++로 개발된 지뢰찾기, 타자게임을 플레이할 수 있는 시스템입니다. 타자게임의 경우 개발에 사용되는 키워드를 활용하여 학습용 게임으로 설계하였으며, 회원 정보 및 게임 스코어는 클라우드 서버(GCP) DB에 저장하여 온라인 스코어 기록을 구현했습니다.

### 🎯 개발 목표

- C++로 지뢰찾기와 타자게임을 구현하여 사용자가 플레이하고 점수를 관리할 수 있는 미니 게임 시스템 개발
- MySQL과 연동하여 프로그래밍 학습을 돕는 게임 플랫폼 구현
- 온라인 스코어 기록 시스템 구축

### 🛠️ 기술 스택

- **Language:** C++
- **Database:** MySQL 8.0.34
- **Cloud:** Google Cloud Platform (GCP)
- **Tools:** Visual Studio Community 2022, GitHub, GitHub Desktop, Discord
- **OS:** Windows

### ✨ 핵심기술 구현

#### DB 설계
- 회원 테이블과 각 게임(지뢰찾기, 타자게임) 스코어 기록 테이블 생성
- 회원의 고유한 ID를 기본키(Primary Key)로 설정하여 식별 및 관계 구성

#### C++ 구현
- ✅ C++ 기반 게임 로직 구현 (지뢰찾기, 타자게임, 점수 계산 및 정렬)
- ✅ 랭킹 점수의 경우, 정렬하여 높은 점수부터 출력
- ✅ 회원 기능 및 점수 기록, 출력 기능 작성
- ✅ 각 화면 인터페이스(UI) 제작
- ✅ 게임 BGM 구축

#### 서버 연동
- ✅ DB 연동 구현 (C++ 전용 MySQL 라이브러리 사용, 회원 정보 및 스코어 기록 저장/조회/수정)
- ✅ Google Cloud Platform DB 구축 및 온라인 스코어 기록 연동

#### 협업 및 테스트
- ✅ 기능 통합 및 테스트 (팀원 코드 병합, 기능별 테스트, 오류 수정)

### 💡 주요 성과

- C++와 MySQL을 처음 연동해보는 과정에서 어려움을 극복하며 프로젝트를 성공적으로 완수
- SW 경진대회에서 **대상 수상**
- 데이터베이스 설계와 백엔드 로직 구현에 대한 깊은 이해 획득

## 🔌 전기차 충전소 예약 시스템

<div align="left">

**📅 개발 기간:** 2025.10.18 ~ 2025.11.18  
**👥 인원 구성:** 2명  
**🎯 담당 역할:** 사용자 인증, 지도(마커, 클러스터), 예약, 결제 (Iamport), UI

### 📝 프로젝트 개요

전기차 충전소 예약·결제·쿠폰·즐겨찾기 기능이 통합된 플랫폼입니다. 카카오맵 API를 활용한 충전소 위치 표시 및 검색, Iamport 결제 연동을 통해 실질적인 문제를 해결하는 전기차 충전소 예약 플랫폼을 개발했습니다.

전기차 사용자는 빠르게 증가하고 있지만 대부분의 충전소는 선착순으로 운영되고 있습니다. 기존 앱은 위치 정보만 제공하며, 예약·결제·쿠폰·즐겨찾기 기능이 통합된 서비스는 검색해봐도 없었습니다. 그리하여, 이러한 실질적인 문제를 해결하는 전기차 충전소 예약 플랫폼을 개발하는 것을 목표로 하였습니다.

### 🛠️ 기술 스택

- **Backend:** Java, Spring Boot
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **API:** 카카오맵 API, Iamport 결제 API
- **Tools:** Git, GitHub

### ✨ 핵심기술 구현

#### 사용자 인증

- ✅ 실시간 상태 (충전중/사용 가능/점검 중)
- ✅ 마커 색상 구분
- ✅ 클러스터링 + 상세 오버레이
- ✅ 필터 (사용 가능 / 예약 가능 / 완속·급속)

#### 충전소 예약
